# Computer-Set-up
Set up for new computers to improve efficiency

## Vim8

### Upgrade to Vim8
`brew install vim --override-system-vim`

### Change .vimrc file
`vim ~/.vimrc`

Paste vimrc.txt contents in .vimrc file

### Install power-line fonts
```# clone
git clone https://github.com/powerline/fonts.git --depth=1
# install
cd fonts
./install.sh
# clean-up a bit
cd ..
rm -rf fonts```
